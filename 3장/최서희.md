# 3장

## 인상 깊은 내용

### 프로덕트의 목적을 달성하기 위한 네 가지 핵심 요소
#### 1. 체계화
데이터서비스실에서 제공하는 데이터 카탈로그의 테이블 메타 정보, 데이터 거버넌스의 비즈니스 용어, 그리고 검증된 데이터 마트를 활용해 데이터 정보의 일관된 체계를 수집한다.
#### 2. 효율화
우아한형제들의 비즈니스를 이해하고, 사내 정보를 쉽고 빠르게 검색할 수 있는 기술을 개발한다.
#### 3. 접근성
누구나 쉽고 빠르게 질문할 수 있도록 웹 기반이 아닌 업무 전용 메신저 슬랙의 대화창을 이용한다.
#### 4. 자동화
데이터 담당자가 없어도 365일 24시간 언제 어디서나 이용할 수 있는 자동화된 데이터 서비스를 지향한다.

## 느낀점

이번 3장을 읽으며 알고리즘 기반의 검색보다 생성형 AI를 활용하는 방식이 훨씬 더 유연한 검색 결과를 낼 수 있다는 것을 체감할 수 있었다.
특히 정해진 규칙을 따르는 것이 아니라 맥락을 이해하고 답을 만들어낸다는 점에서, 생성형 AI는 기존 검색 방식과는 다른 가능성을 보여준다.
AI를 활용해 업무 효율을 높이는 데에는 사실상 한계가 없다고 느껴졌고, 앞으로 이를 어떻게 활용하느냐에 따라 개인의 생산성 차이는 더욱 커질 것이라는 생각이 들었다.
